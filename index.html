ByteBank - Digital Expense Analyzer

About

This is a console-based Python application to manage personal expenses. You can:

  • Add, update, and delete expenses
  • Filter expenses by category or date
  • View reports, totals, and summary dashboards
  • Export expenses to CSV for further analysis

Run Instructions

Since this is a Python console application, follow these steps:

  1. Clone the repository.
  2. Install Python or use a mobile Python app (Pydroid 3 / Pythonista).
  3. Run main.py to start the application.
